📅 BookEase – Appointment Booking System

BookEase is a full-stack MERN appointment booking application that allows users to book appointments and enables admins to manage bookings through a dashboard.

Built as my first complete MERN Stack project and deployed successfully.

---

📹 Demo Video

https://appointment-booking-app11.vercel.app
---

Features

User Features

- Book appointments
- Select available time slots
- Prevent double booking
- View booked appointments using email
- Responsive UI

Admin Features

- View all appointments
- Update appointment status
- Delete appointments
- Filter appointments
- Calendar-based appointment view
- Dashboard statistics

---

Frontend

Backend

Database
